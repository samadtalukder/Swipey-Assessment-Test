# Swipey-Assessment-Test
The repository has contains the Swipey Assessment Test, where an Android project is created and implement their task

### Built With

* Android Studio 
* Build gradle 7.2.0
* Android SDK 32
* Kotlin Version 1.6.21

## Language

* Kotlin

## Libraries Used

* AndroidX
* LiveData
* ViewModel
* LifeCycle
* Navigation UI
* Retrofit2
* OkHttp3
* Gson
* Hilt
* Glide
